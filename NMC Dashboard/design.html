5. AI Logic (rule-engine, no subscription)

5.1 Forward Department / Responsible Team

FO / Fiber / NTTN / Last mile       → NCSS
NTTN capacity / NTTN end / SCL/F@H   → Survey & Transmission
Telco POP / BL POP / GP POP / STL    → Survey & Transmission
BTS down / Power / Surecom           → BTS & Power Infrastructure
Router / Switch / IIG / Upstream / PNI / NIX / Traffic / BGP / OSPF → NGNC
BRAS / Dist BRAS / OLT / ONU (logical) / PON → BNOC
IPTSB / I&I / Other                  → Manual select

5.2 AI Chatbox

  • Parses pasted ticket text → fills form automatically
  • Suggests Category, Sub-Category, Department
  • Learns from manual overrides (stored in aiTraining localStorage)
  • Suggest contact by fuzzy match on BTS/Area string

5.3 Contact AI

  • Input: free text (e.g. "CTG", "DHK-Dhanmondi", "BL_Sonagazi")
  • Output: ranked list — exact match > zone match > substring > AI learned
  • User feedback (👍/👎) reinforces the model

7. Mail Templates

  • NTTN Query (SCL/F@H via portal — auto-prefilled URL; other NTTN — mailto: with body)
  • IIG Down / Traffic Fall (laser + ping + cacti graph attach note)
  • Telco POP Mail (GP/BL/STL — power check)
  • BRAS Bandwidth Zonewise (Night — 3 Excel exports)
  • Weekly Report (auto Sun night)
  • Monthly Report (auto last day night)

8. CCB / NCR / PID Logic

  1. Outlook mailbox → IMAP/Graph polling (optional add-on), or manual add with parsed fields
  2. Dashboard shows items whose now < end + status Active
  3. When now > end: status → Expired, banner asks "Still ongoing?" → Yes reopens, No closes
  4. Expired items go to history, available in Reports → "CCB/NCR Log"

9. WhatsApp Sharing

Use https://wa.me/<group-invite>?text=<encoded> link to push to a group. For automation (wa-automate / open-wa) the portal exposes a "Send to WhatsApp" button that opens wa.me with pre-filled text. Optional backend worker can call a self-hosted wa-bot. No subscription required for the link-share model.

11. Open the live portal

Open index.html in any modern browser (Chrome/Edge). All data persists in localStorage. To go live, host the folder on IIS / nginx / python http.server — no backend, no subscription.
